# ALBUMLIN - Typecho 记忆相册插件

一个优雅的 Typecho 相册插件，自动从文章中提取图片，支持背景音乐和灯箱预览。

## 功能特性

- 📸 **自动提取图片** - 从指定分类的文章中自动提取 Markdown 图片
- 🎵 **背景音乐** - 内置音乐播放器，支持自定义音乐链接
- 🔍 **灯箱预览** - 点击缩略图全屏预览，支持键盘导航和触摸滑动
- 📱 **响应式设计** - 完美适配桌面端和移动端
- ⚡ **懒加载** - 图片懒加载优化性能

###激活插件
1. 登录 Typecho 后台
2. 进入 控制台 → 插件
3. 找到 AlbumGallery 并点击 启用
4. 点击 设置 配置插件参数
5. 分类id  后台找到你的分类,点击分类,查看搜索框链接,链接最后的数字是你这个分类的id

访问相册
启用插件后，直接访问：
https://你的域名/album

Q: 启用插件后显示 Server Error？
A: 请检查：
1. 
PHP 版本是否 >= 7.4
2. 
文件夹名是否为  AlbumGallery （严格区分大小写）
3. 
文件编码是否为 UTF-8 无 BOM
Q: 相册页面空白或没有图片？
A: 请检查：
1. 
分类 ID 是否正确
2. 
该分类下是否有已发布的文章
Q: 如何修改相册样式？
A: 编辑  views/album.php  文件中的 CSS 样式，或添加自定义 CSS。
