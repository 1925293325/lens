# LENS  - 相册插件

一款极简优雅的 Typecho 相册插件，采用纯白顶栏 + 相机首图风格，支持三列/双列网格切换，完美适配移动端。

## 特性

- 📷 **复古相机首图** - 支持后台自定义顶部背景图
- 🎨 **纯白极简风格** - 白色顶栏 + 黑色文字，经典耐看
- 🔲 **双模式网格** - 三列紧凑/双列大图，后台一键切换（含移动端）
- 👆 **手势操作** - 支持左右滑动切换、双击缩放、键盘导航
- 🎬 **沉浸预览** - 点击缩略图全屏预览，底部带缩略图列表
- 🚀 **无感加载** - 图片懒加载，无限滚动体验
- 📱 **完美移动端** - 响应式布局，手机端也能用三列模式

## 安装

1. 下载本插件
2. 将文件夹重命名为 `LENS `
3. 上传至 `/usr/plugins/` 目录
4. 后台启用插件
5. 访问 `/lens` 即可查看相册

## 配置说明

后台设置项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **相册标题** | 显示在顶栏和首图左下角 | 我的相册 |
| **相册副标题** | 首图左下角小字（英文） | Scenery along the way |
| **默认分类ID** | 从哪个分类提取文章图片 | 2 |
| **首图背景** | 相册页面顶部大图 URL | Unsplash 相机图 |
| **网格布局** | 三列（小图）或双列（大图） | 三列 |

### 首图背景推荐

默认使用 Unsplash 复古相机图，如需更换：
请将相机图片上传到 usr/plugins/LENS/img/ 文件夹并命名为 camera-bg.jpg，或填写完整URL如 https://你的图床.com/photo.webp

Q: 启用插件后显示 Server Error？
A: 请检查：
1. 
PHP 版本是否 >= 7.4
2. 
文件夹名是否为  AlbumGallery （严格区分大小写）
3. 
文件编码是否为 UTF-8 无 BOM
Q: 相册页面空白或没有图片？
A: 请检查：
1. 
分类 ID 是否正确
2. 
该分类下是否有已发布的文章
Q: 如何修改相册样式？
A: 编辑  views/lens.php  文件中的 CSS 样式，或添加自定义 CSS。
