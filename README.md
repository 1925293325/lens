# AlbumGallery - Typecho 记忆相册插件

一个优雅的 Typecho 相册插件，自动从文章中提取图片，支持云存储缩略图、背景音乐和灯箱预览。

## 功能特性

- 📸 **自动提取图片** - 从指定分类的文章中自动提取 Markdown 图片
- ☁️ **云存储支持** - 支持七牛云、又拍云、阿里云 OSS 图片缩略图
- 🎵 **背景音乐** - 内置音乐播放器，支持自定义音乐链接
- 🔍 **灯箱预览** - 点击缩略图全屏预览，支持键盘导航
- 📱 **响应式设计** - 完美适配桌面端和移动端
- ⚡ **懒加载** - 图片懒加载优化性能

###激活插件
1. 登录 Typecho 后台
2. 进入 控制台 → 插件
3. 找到 AlbumGallery 并点击 启用
4. 点击 设置 配置插件参数

访问相册
启用插件后，直接访问：
https://你的域名/album

Q: 启用插件后显示 Server Error？
A: 请检查：
1. 
PHP 版本是否 >= 7.4
2. 
文件夹名是否为  AlbumGallery （严格区分大小写）
3. 
文件编码是否为 UTF-8 无 BOM
Q: 相册页面空白或没有图片？
A: 请检查：
1. 
分类 ID 是否正确
2. 
该分类下是否有已发布的文章
3. 
文章 Markdown 中是否包含图片
Q: 云存储缩略图不生效？
A: 请检查：
1. 
域名配置是否正确（不包含 http://）
2. 
图片 URL 是否包含配置的域名
3. 
云存储是否开启图片处理功能
Q: 如何修改相册样式？
A: 编辑  views/album.php  文件中的 CSS 样式，或添加自定义 CSS。
